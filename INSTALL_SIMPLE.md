## В МАГАЗИНЕ — УСТАНОВКА

### Шаг 1. Открыть программу для ввода команд

**На Mac:**
- Нажми Command + Space (одновременно)
- Напиши `Terminal` (английскими буквами)
- Нажми Enter
- Откроется чёрное окно (терминал)

**На Windows:**
- Нажми кнопку Windows (внизу слева)
- Напиши `PowerShell` 
- Нажми Enter
- Откроется окно с голубым фоном

### Шаг 2. Установить Python

Зайди на **python.org/downloads** через браузер на том же ноуте.

Нажми **Download Python 3.12** (жёлтая кнопка).

Запусти файл, который скачается. На Windows **обязательно** отметь галочку **"Add Python to PATH"** в первом окне.

После установки закрой браузер и терминал полностью. Открой терминал заново.

Напиши в терминал:
```
python3 --version
```

Нажми Enter. Должна написать: `Python 3.12.x` (какая-то цифра вместо x).

Если написало "command not found" — позвони мне, что-то не так.

### Шаг 3. Установить Git

**На Mac:**
- В терминале напиши: `git --version`
- Нажми Enter
- macOS спросит "установить?", нажми "Install"

**На Windows:**
- Открой браузер → **git-scm.com**
- Нажми "Download for Windows"
- Запусти файл
- Во всех окнах нажми "Next" (дальше), потом "Install"

### Шаг 4. Скачать нашу программу

В терминале напиши (скопируй и вставь — Ctrl+V):

```
cd ~
git clone https://github.com/isaevsaf-byte/havas-pilot.git
cd havas-pilot
```

Нажми Enter после каждой строки.

Если попросит логин GitHub — спроси у Сафара (ему нужен Personal Access Token).

### Шаг 5. Установить библиотеки (самая долгая часть — 15 минут)

В терминале:
```
pip3 install -r requirements.txt
```

Нажми Enter и жди. Будет много текста, это нормально. Жди до конца, пока не напишет что-то вроде "Successfully installed".

### Шаг 6. Прописать ключи (ВСЕ ПАРОЛИ ХРАНЯТСЯ ЛОКАЛЬНО НА НОУТЕ)

**На Mac:**

В терминале напиши:
```
nano ~/.zshrc
```

Нажми Enter. Откроется текстовый редактор. Прокрутись вниз (стрелочка вниз) и в конец добавь эти три строки:

```
export SUPABASE_URL="https://kxyyvnxklbczuofzaoow.supabase.co"
export SUPABASE_KEY="sb_publishable_79gUEoh_qoocFxnwied-Tw_IE3uU00j"
export CAMERA_URL="rtsp://ЛОГИН:ПАРОЛЬ@IP:554/stream1"
```

Вместо:
- ЛОГИН → напиши что дали установщики (типа admin)
- ПАРОЛЬ → их пароль
- IP → их IP (типа 192.168.1.64)

Пример готово:
```
export CAMERA_URL="rtsp://admin:qwerty123@192.168.1.64:554/stream1"
```

После этого:
- Нажми Ctrl+O (одновременно)
- Нажми Enter (подтвердить сохранение)
- Нажми Ctrl+X (выход)

Потом в терминале:
```
source ~/.zshrc
```

Нажми Enter.

**На Windows:**

В PowerShell копируй и вставь эту одну огромную строку (одна команда):

```powershell
[Environment]::SetEnvironmentVariable("SUPABASE_URL","https://kxyyvnxklbczuofzaoow.supabase.co","User"); [Environment]::SetEnvironmentVariable("SUPABASE_KEY","sb_publishable_79gUEoh_qoocFxnwied-Tw_IE3uU00j","User"); [Environment]::SetEnvironmentVariable("CAMERA_URL","rtsp://ЛОГИН:ПАРОЛЬ@IP:554/stream1","User")
```

Вместо ЛОГИН/ПАРОЛЬ/IP вставь реальные значения.

Пример:
```powershell
[Environment]::SetEnvironmentVariable("CAMERA_URL","rtsp://admin:qwerty123@192.168.1.64:554/stream1","User")
```

Нажми Enter.

Закрой PowerShell полностью и открой заново.

## В МАГАЗИНЕ — ПРОВЕРКА

### Проверка 1. Библиотеки встали

В терминале:
```
python3 check_imports.py
```

Нажми Enter.

Должно написать пять раз "OK: ..." (detector, tracker, reid, database, config).

Если есть FAIL → проблема с установкой. Позвони.

### Проверка 2. Интернет есть и база работает

В терминале:
```
python3 setup_supabase.py
```

Должно написать "Supabase готов".

Если ошибка → интернета нет или ключи неправильные. Проверь Wi-Fi.

### Проверка 3. Программа в принципе работает

В терминале:
```
python3 test_with_video.py
```

Это тестирует программу на заготовленном видеофайле. Займёт минуту.

В конце должно написать что-то вроде:
```
Всего событий: 1
Новых визитов: 1
```

Если что-то упадёт → позвони.

### Проверка 4. Камера отвечает

В терминале напиши:
```
python3 -c "import cv2, os; cap = cv2.VideoCapture(os.environ['CAMERA_URL']); print('ОК' if cap.read()[0] else 'КАМЕРА НЕ ВИДНА')"
```

Нажми Enter.

Если напечатает "ОК" → супер, камера видна.

Если "КАМЕРА НЕ ВИДНА" → спроси установщиков: 
- Ноут и камера в одной сети (Wi-Fi)?
- IP правильный?
- Логин и пароль правильные?

## В МАГАЗИНЕ — БОЕВОЙ ЗАПУСК

### Первый запуск — С ОКНОМ (чтобы проверить)

В терминале:
```
python3 main.py
```

Нажми Enter.

Должно открыться окно с видео с камеры. Проверь:

1. **Люди видны в рамочках?** Если да → детектор работает.
2. **Жёлтая линия посредине видео?** Если да → всё хорошо.
3. **Линия проходит ТАМ, ГДЕ ЛЮДИ РЕАЛЬНО ВХОДЯТ?** 
   - Если линия слишком высоко → отредактируй `config.py` строку `LINE_POSITION = 0.7` (выше = 0.3, ниже = 0.9)
   - Если правильно → ок
4. **Зайди сам перед камерой 2 раза** (туда-обратно):
   - Первый раз в терминале должен написать: `[время] IN | new | visitor_...`
   - Второй раз (когда вернулся): `[время] OUT | repeat | visitor_...`

Если всё совпадает → **система работает!**

Выход из программы: нажми клавишу `q` (и Enter).

### Второй запуск — БЕЗ ОКНА (постоянная работа)

Это на постоянку, когда ноут стоит и никто не смотрит.

**На Mac:**
```
HEADLESS=1 python3 main.py
```

**На Windows PowerShell:**
```
$env:HEADLESS="1"; python main.py
```

Нажми Enter. Окна не будет, в терминале будут печататься события типа `[11:23:45] IN | new | ...` когда люди входят.

Выход: Ctrl+C.

### Проверить дашборд

Открой браузер → зайди в **Streamlit Cloud** (ссылка в README.md).

Через 5 минут работы в дашборде должен загореться статус 🟢 "Система работает" и появляться события (таблица внизу).

## ЕСЛИ ЧТО-ТО СЛОМАЛОСЬ

### Ошибка: "command not found: python3"
Python не установлен или путь не прописан. Скачай с python.org заново, **галочку "Add to PATH" не забудь**.

### Ошибка: "No module named ..."
Не все библиотеки встали. Запусти ещё раз:
```
pip3 install -r requirements.txt
```

### Окно не открывается / "display not found"
Запускай с флагом:
```
HEADLESS=1 python3 main.py
```

### Камера не отвечает
- Ноут в той же Wi-Fi что камера?
- IP правильный? (спроси установщиков ещё раз)
- Пароль не с опечатками?

### В дашборде нет событий
- Ноут может отключиться от интернета. Проверь Wi-Fi.
- Программа может упасть. Запусти `python3 main.py` заново.
